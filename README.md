- 🔤 String manipulation utilities
- 🧠 Logical problem-solving exercises
- 🧩 Beginner to intermediate coding patterns

## Critical Mentor Feedback

Right now this repo is:

> ❌ Just a collection of files  
> ✅ Not a system

---

### What you're missing (important)

You're solving problems, but not designing software.

---

### What a strong engineer would do

Instead of:
```

PrimeNumber.java
Factorials.java

````

You should think:

```java
class NumberUtils {
    static boolean isPrime(int n) {}
    static long factorial(int n) {}
}
````

---

### Why this matters

Because in real-world systems:

* You don’t write 100 classes
* You design reusable modules
