# ☕ Java Programs Collection

A curated collection of Java programs focused on building strong fundamentals in problem-solving, logic building, and core programming concepts. This repository is structured to help learners practice commonly asked coding problems in interviews and exams.

---

## 🚀 Features

- 🔢 Number-based programs (Prime, Factorial, Strong Number, etc.)
- 🔤 String manipulation utilities
- 🧠 Logical problem-solving exercises
- 🧩 Beginner to intermediate coding patterns
- 📂 Organized folder structure
